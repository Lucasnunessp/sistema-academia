# Sistema de Academia em Python

## 📌 Objetivo
Sistema para gerenciamento de alunos, planos e pagamentos de uma academia.

## ⚙️ Funcionalidades
- Cadastro de alunos
- Cadastro de planos
- Registro de pagamentos
- Listagem de dados

## ▶️ Como executar
1. Abra o terminal
2. Vá até a pasta do projeto
3. Execute:

python main.py
